# sqlx example

Searchable encryption with CipherStash Proxy, Go, and sqlx.

## Quickstart

Requirements:

 - Golang 1.23.2
 - Docker

``` bash
# Clone the repo
git clone https://github.com/cipherstash/encrypt-query-language

# Change into example directory
cd encrypt-query-language/languages/go/examples/sqlx

# Start the database
docker-compose up

# Run the examples
make
```
